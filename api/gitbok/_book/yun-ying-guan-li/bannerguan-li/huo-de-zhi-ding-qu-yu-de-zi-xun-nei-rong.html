获得指定区域的资讯内容

POST

admin/operate/getIndexBanner

参数

字段 类型 描述
regionId Number 指定区域的id
  • Success-Response:
HTTP/1.1 200 OK
{
"code": 0,
"msg": "",
"data": {
         {
             "region_name":"text",
             "zx_id":[
                 {
                     "id":"xxx",
                     "weight":"xxxxxxxxxxxx",
                     "zx_name":"front/test/test",
                     "information_type":"xxxxxxxxxxxx",
                     "create_time":"xxxxxxxxxxxx"
                 }
             ]
         }
  }
}

Error 4xx

名称 类型 描述
error Object[] 这里是失败时返回实例
  • Error-Response:
HTTP/1.1 40x
{
  "code": "1",
   "msg": '请求失败'
}

 HTTP/1.1 5xx
{
  "code": "2",
   "msg": '请求方式错误'
}
